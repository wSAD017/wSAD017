Formal greetings

This type of greetings, you can use them in more specific and less casual occasions than usual.


Hello: Hola

Good morning: Buenos días

Good afternoon: Buenas tardes (entre las 12:00 pm y las 5:00 pm)

Good evening: Buenas tardes (entre las 6:00 pm y las 8:00 pm)

How are you?: ¿Cómo estás?

Good to see you: Que bueno verte

Nice to see you: Que agradable verte


Informal greetings

Hi: Hola

Hey!: Hola

How’s it going?: ¿Cómo te ha ido?,¿Cómo te está yendo?

How are you all?: ¿Cómo están todos?

How are you doing?: ¿Cómo vas/van?


What was learned in Session 02


[h1 - h6]:

  • Spanish Estos cumplen la función de crear titulos-suptitulos, los cuales te permiten organizar de una mejor forma las paginas web. Existen barios tamaños siendo h1 el mas grande y h6 el mas pequeño
  • EnglishThese fulfill the function of creating titles-suptitles, which allow you to organize in a better way the web pages. There are several sizes being h1 the biggest and h6 the smallest.

[p]:

  • SpanishCumple la función de agregar un párrafo y/o texto deseado.
  • EnglishIt fulfills the function of adding a paragraph and/or desired text.

[pre]:

  • SpanishPermite agregar un texto al igual que [p] con la diferencia de que las modificación realizada desde HTML serán tomadas en cuenta.
  • EnglishAllows you to add a text like [p] with the difference that the modifications made from HTML will be taken into account.

[hr]:

  • SpanishPermite añadir una linea de separación entre los textos.
  • EnglishAllows you to add a separator line between texts.

[br]:

  • SpanishInserta un espaciado, que se aplica entre los textos.
  • EnglishInserts a spacing, which is applied between texts.

[title]:

  • SpanishDefine el que será el titulo principal de la pagina web.
  • EnglishDefine what will be the main title of the web page.

[img]:

  • SpanishPermite añadir una imagen de tu preferencua a la pagina.
  • EnglishAllows you to add an image of your choice to the page.

[style]:

  • SpanishPermite añadir características de estilo a la mayoría de formatos mencionados anteriormente.
  • EnglishAllows you to add styling features to most of the formats mentioned above.
